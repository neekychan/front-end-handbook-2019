# 2.10 前端薪酬
<!-- The national average in the U.S for a mid-level front-end developer is somewhere between $65k and 100k. -->
美国全国平均水平，一个中级前端开发者的薪酬在6.5万美金到10万美金之间。

当然，对于除此进入前端领域的新人来说，这取决于你工作的地点和经验。
