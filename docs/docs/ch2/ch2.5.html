2.5 前端开发者对...开发

2.5 前端开发者对...开发

前端开发者使用HTLM，CSS和JS，通常运行在以下操作系统的网页平台上：

  • Android
  • Chromium
  • iOS
  • OS X(例如： MacOS)
  • Ubuntu(或某种Linux)
  • Windows

通常这些操作系统运行在以下一种或多种设备上：

  • 桌面电脑
  • 手提电脑 / 上网本
  • 移动电话
  • 平板电脑
  • 电视
  • 手表
  • 物件（例如：任何能想象的东西，汽车、冰箱、电灯、恒温器等等）

Image source: https://www.enterpriseirregulars.com/104084/roundup-internet-things-forecasts-market-estimates-2015/

普遍来说，前端技术能够运行在上述操作系统和使用以下运行时的网页平台设备中：

  • 网页浏览器（例如：Chrome，IE，Safari，Firefox）
  • 无头浏览器（例如：Headless Chromium）
  • WebView/浏览器标签（iframe）被嵌入在原生应用中，桥接原生API和运行时。WebView应用通常包含由网页技术构造的UI。（例如：HTML，CSS和JS）（例子：Apache Cordova, NW.js, Electron） 由网页技术构建的原生应用，在运行时通过原生API的桥接进行解释。UI将使用原生UI部分（例如：iOS原生控制）而非网页技术。（例子：NativeScript，React Native）
